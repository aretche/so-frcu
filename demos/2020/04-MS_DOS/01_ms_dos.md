## Uso de Memoria

```
MEM
```

Muestra programas TSR y los clasifica
```
MEM /C /P
```
Muestra memoria libre
```
MEM /F
```

Muestra detalle de todo el uso de memoria
```
MEM /D /P
```

Muestra el uso de memoria de algún modulo en particular
```
MEM /M MSDOS
```

```
MEM /M COMMAND
```
