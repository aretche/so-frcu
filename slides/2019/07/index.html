MacOS Sistema 2

Introduced multiple folders, the "Shut Down" command, and a quick-launching application feature.

Added support for AppleTalk (networking) and the newly introduced LaserWriter to use it. Also introduced the HFS (Hierarchical File System).

MacOS Sistema 3

Implemented HFS, support for several new technologies including SCSI and AppleShare, and Trash "bulging“.

MacOS Sistema 4

Software for improved hardware - expansion slots, the Apple Desktop Bus (ADB), internal hard drives.

MacOS Sistema 5

Added MultiFinder which allowed cooperative multitasking, so time was given to the background applications only when the running application yielded control.

System Software 5 is the first Macintosh operating system to be given a unified "Macintosh System Software" version number

MacOS Sistema 6

Added MacroMaker allowed users to record mouse and keyboard input as "macros."

A maximum of 15 Desk Accessories could be installed at one time, including the Chooser, Scrapbook, and Control Panel.

MacOS Sistema 7

Added virtual memory, personal file sharing, QuickTime, QuickDraw 3D, and improved UI.

Used over a megabyte (Sys 6 took up 600K)

It was the first Apple operating system to be available on CD, although it shipped on a set of 15 floppy disks initially.

MacOS Sistema 8

Improved multi-tasking, files can be copied in the background, GUI skins introduced.

Initially planned as Mac OS 7.7, renumbered "8" to exploit a legal loophole and terminate third-party manufacturers' licenses to System 7
